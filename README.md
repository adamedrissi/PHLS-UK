## Problem Statement
NHS services in the UK are increasingly overburdened, leading to long waiting times and limited appointment availability. As a result, more patients turn to private healthcare for quicker access, but the private sector is fragmented and difficult to navigate. Patients struggle to compare providers based on budget, location, specialty, and availability, while existing platforms often lack transparency or comprehensive filtering. At the same time, private doctors and clinics lack a centralised space to promote their services and publish up-to-date schedules. There is a need for a clear, accessible platform that matches patients with suitable private healthcare options efficiently.

- **Frontend**: VS Code - JavaScript, React Vite jsx (CSS for styling)
- **Database**: pgAdmin4, PostgreSQL
- **Build Tools**: Maven for Backend, npm for Frontend
- **Deployment**: Railway (Trial) for Backend and Database, Netlify (Free) for Frontend
